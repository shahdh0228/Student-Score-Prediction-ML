# 🎓 Student Score Prediction - ML Internship Task

## 📌 Objective
Predict students' exam scores using academic and behavioral features.

## 📊 Models Implemented
- Linear Regression (Baseline)
- Polynomial Regression (Degree 2)
- Feature Engineering Experiments

## 📈 Results
|         Model         |  R²  |
|-----------------------|------|
| Linear Regression     | 0.77 |
| Polynomial Regression | 0.76 |

The dataset shows a strong linear relationship. Additional complexity did not significantly improve performance.
